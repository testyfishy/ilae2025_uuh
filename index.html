Project Overview

This project develops an ILAE 2025-compliant seizure classification system for integration into Epic EHR as a SmartForm. By embedding validated classification logic into routine clinical documentation, we aim to improve classification accuracy, generate research-ready patient cohorts, and establish infrastructure for multi-center collaboration.

  • Aim 1: SmartForm Development & Validation
  • Aim 2: Multi-Semiology Documentation Framework
  • Aim 3: Clinical Integration via SmartPhrase
  • Aim 4: Cohort Database & Metadata Generation
